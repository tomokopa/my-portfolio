洗濯物外干し時の失敗を減らす！神戸駅の天気予報LINEアカウント

2021年8月3日作成

晴れた日の洗濯って、なんて気持ち良いんだろう！
ベランダでの洗濯を終え、さわやかな気分で出勤したあなた。

しかし午後、まさかの土砂降り。せっかく外干しした洗濯物がビショビショでショック。
こんな時に限って、明日以降ずっと雨の予報でさらに落ち込んでしまった…

そんな経験はありませんか？

「Laundry Weather KOBE」なら、外干しの失敗を大幅に減らせます。
毎朝7時頃、外干しについてのシンプルな天気予報のLINEメッセージをあなたにお届けするからです。
このサービスはLINEで「Laundry Weather KOBE」アカウントを友だち登録するだけで、すぐにご利用いただけます。

サービス概要

このサービスは洗濯に特化したシンプルな天気予報LINEアカウントです。
「外干しした洗濯物がよく乾くかどうか」ではなく、「外干しして失敗するか」の観点からまとめた情報をお届けします。
天気予報の対象地点は神戸駅のみです。

利用方法・停止方法

上記LINEアカウント「Laundry Weather KOBE」を友だち登録すると、毎朝7時頃にこのアカウントからメッセージが届きます。
サービスを停止したい場合は、このLINEアカウントのトーク画面右上のメニューからブロックをしてください。

こんな方におすすめ

洗濯物はなるべく日中に外干ししたいが、失敗したくないという方。
日中外出する方にも、主婦や在宅ワークの方にもおすすめです。
ただし、夜間の外干しについてはサービスの対象外です。

サービス詳細

本日の外干しアドバイス

メッセージが届いた日の7時～18時の1時間ごとの天気予報です。
天気予報は以下の3段階でお知らせします。

  1. OK
    下記2つに該当しません。外干しして失敗する可能性は低いです。
  2. 雨等悪天候
    雨や雪が降るなど、洗濯物に悪影響が出る可能性が高いです。
  3. 風速10m/s超
    天気は晴れまたは曇りですが、平均風速が10m/sを超えています。
    洗濯物が飛ばされたり、埃まみれになるなど、悪影響が出る可能性が高いです。

参考：気象庁「風の強さと吹き方」

【この天気予報の活用方法】
メッセージが届いた日に洗濯物を外干しするかどうかの参考にしてください。

明日以降の外干しアドバイス

メッセージが届いた日の翌日を含む4日間の、9時、12時、15時、18時の天気予報です。
天気予報は「本日の外干しアドバイス」と同じ法則でお知らせします。

【この天気予報の活用方法】
明日以降の天気をふまえて、メッセージが届いた日に洗濯物を外干しするかどうかの参考にしてください。
例えば、明日以降ずっと悪天候が続くのであれば、今日洗濯した方が良いかもしれません。

サービス作成理由

  • 洗濯物の外干しに特化したシンプルな天気予報が欲しかったから
  • 毎日同じ時間に誰かから天気予報を確実に教えて欲しかったから
    （テレビの天気予報や天気予報アプリを見忘れることが多いから）
  • 独立したアプリではなく、使い慣れたツール（今回はLINE）で気軽に情報を受け取りたかったから
  • スクレイピングの参考書で学んだWeb APIを活用してみたかったから

苦労した点・こだわりの点

視覚に訴えるシンプルさの追求

天気の判定を「OK」「雨等悪天候」「風速10m/s超」の3つに絞りました。
これにより、一目で「OK」か否かの判断が可能です。
また、時間帯を日中に絞ったことで、メッセージの冗長さを防ぎました。

試験運用と改善（コード含む）

約1か月間の試験運用により、見にくい要素と不要な要素の発見をしました。
改善する際、関数を取り入れることで、コードもシンプルにしました。

GitHubからのデプロイ

コードの公開をしたかったため、GitHubからHerokuへデプロイしました。
当初、ローカルのgitリポジトリからheroku createとgit push heroku masterのコマンドでHerokuへデプロイしていました。
しかし、この方法ではおそらくGitHubのように手軽にコードを公開することが出来ないと思われました。

使用技術と選定理由

Python 3.9

Pythonは初めて本格的に学習した言語であり、知識があるため。
Pythonを学習した理由は、

  • 初心者でも学びやすい、という情報をインターネットから得たこと
  • 機械学習に興味があったこと

です。
バージョンは、Pythonanywhereが対応しているバージョンの中で開発時に最新だった3.9を選定しました。

Heroku

毎日同じ時間にWeb上でプログラムを動かす方法を学ぶ際に参考にしたホームページで使用されていたため。

LINE Messaging API

誰でも簡単に使えるサービスにしたかったため。
LINEの友達登録だけでメッセージが届くようになる、という手軽さに魅力を感じました。

OpenWeatherMap のWeather API

スクレイピングの参考書で学習したため。

GitHub

コードを公開するため。
Herokuを利用してWebアプリを公開するだけならGitHubを利用しない方法もありますが、あえて利用しました。
